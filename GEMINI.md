# Projekt: Kurs-Plattform
Dieses Projekt ist ein interaktives Learning Portal für Gemini CLI.

##  Geplante Features
* Lernpfade, Module, Kurse, Codelabs und Tutorials.
* Interaktiver Code-Editor (Monaco Editor).

## Referenzen & Stil
* **Design-Vorbilder:** Microsoft Learn, W3Schools, Google Codelabs.
* **Tech-Stack:** HTML, CSS, JavaScript (Vanilla JS).

##  Editor-Setup
* Monaco Editor v0.55.1
* monaco-languageclient

##  Navigation & Benutzerdaten
* **Impressum:** Nutze `impressum.md` im Hauptverzeichnis. Erstelle dafür einen Menüpunkt "Impressum".
* **Datenschutz:** Erstelle eine `datenschutz.md` und den dazugehörigen Menüpunkt "Datenschutz".
* **Module:** Alle verfügbaren Lernmodule sollen in der Navigation angezeigt werden.
